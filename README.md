# interactive-form-v3
 1) All the input text fields required and represented by asterik sign "*" have real time error validations. Until the field is filled out correctly, the hint and red warning sign will be displayed. If pass the green check mark sign will be displayed. 
 
 2) The name field has conditional error message where if user have it blank it will show "Name field cannot be blank". If it's not filled correctly it will show "Must include first and last name"
